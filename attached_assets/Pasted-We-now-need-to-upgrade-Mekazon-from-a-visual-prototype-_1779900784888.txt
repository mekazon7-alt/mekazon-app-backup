We now need to upgrade Mekazon from a visual prototype into a real Shopify-connected mobile app architecture.

CURRENT BUSINESS:
Mekazon website is built on Shopify:
https://www.mekazon.com

The app must be connected to the existing Shopify backend so that:
- When we add a product on Shopify, it can appear in the app automatically
- When we add products to a country collection on Shopify, the correct country version of the app updates automatically
- When we add products to Shopify categories/collections, the app can use those collections for app sections
- Shopify remains the source of truth for products, prices, inventory, collections, variants, cart, and checkout

IMPORTANT:
Do not build this as a fake standalone grocery app.

CATEGORY STRUCTURE:
The app currently uses nice labels like:
- Fresh
- Staples
- Sauces
- Spices
- Meal Inspiration
- Made for Uganda/Kenya/Ethiopia

But Shopify website collections may have different names.

So create a mapping layer where app category names can map to Shopify collection handles.

Example:
App label: “Fresh”
Shopify handle: “fresh-vegetables” or “uganda-fresh”

App label: “Staples”
Shopify handle: “flour-rice-staples” or “maize-flour”

This way the app design can stay beautiful even if Shopify collection names are different.
